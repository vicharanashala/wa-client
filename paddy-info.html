1.Data Storage Architecture

The NPK data is sourced from Google Sheets containing crop-specific nutrient requirements for each Indian state. This data is stored in MongoDB for efficient querying and retrieval.

1.1 MongoDB Collection Name

Recommended Collection Name
crop_nutrient_requirements — Stores NPK data indexed by state and crop for fast lookups

1.2 Data Source Structure

Google Sheets contain one sheet per state. Each sheet follows a consistent structure with crop data, nutrient requirements, and source documentation. States include:

Andaman and Nicobar Islands, Andhra Pradesh, Arunachal Pradesh, Assam, Bihar, Chandigarh, Chhattisgarh, Dadra and Nagar Haveli and Daman, Delhi, Goa, Gujarat, Haryana, Himachal Pradesh, Jammu and Kashmir, Jharkhand, Karnataka, Kerala, Ladakh, Lakshadweep, Madhya Pradesh, Maharashtra, Manipur, Meghalaya, Mizoram, Nagaland, Odisha, Puducherry, Punjab, Rajasthan, Sikkim, Tamil Nadu, Telangana, Tripura, Uttarakhand, Uttar Pradesh, West Bengal

1
Google Sheets
State-wise sheets with crop NPK data
2
Data Parsing
Extract N, P, K values and source references per crop
3
MongoDB Storage
Store in crop_nutrient_requirements collection
4
Indexed Queries
Fast lookup by state + crop combination

3.Query Type 1: Required NPK Lookup

When a farmer knows their crop and location but needs to know the recommended NPK values for optimal growth.

3.1 Query Pattern

Searching Parameters
Crop: Name of the crop to grow
Location: State and optionally district/region

3.2 Example Query & Response

Sample Conversation
Q: I want to grow wheat in Kathua, Jammu and Kashmir, tell me the required NPK values?

A: Based on the provided sources, the required NPK values for growing wheat in Kathua, Jammu and Kashmir are:

Nitrogen (N): Required amount is 60 kg/acre.
Source: ICAR - Directorate of Wheat Research (Page 8)

Phosphorus (P): Required amount is 24 kg/acre.
Source: ICAR - Directorate of Wheat Research (Page 8)

Potassium (K): Required amount is 16 kg/acre.
Source: ICAR - Directorate of Wheat Research (Page 8)

3.3 System Flow

1
Parse Query
Extract crop (wheat) and location (Jammu and Kashmir)
2
Normalize Inputs
Convert to lowercase, match against database
3
Database Query
db.crop_nutrient_requirements.findOne({state, crop})
4
Format Response
Return NPK values with sources and page numbers

4.Query Type 2: NPK Deficit/Surplus Analysis

When a farmer has soil test results and wants to know how much fertilizer to add for their chosen crop. The system calculates deficit/surplus and provides fertilizer recommendations.

4.1 Query Pattern

Searching Parameters
Crop: Name of the crop to grow
Location: State/region
Current Soil NPK: N, P, K values from soil test (kg/acre)

4.2 Common Fertilizers (More to be added)

Fertilizer N (%) P₂O₅ (%) K₂O (%)
Urea (CH₄N₂O) 46% — —
Single Superphosphate (SSP) — 16% —
Muriate of Potash (MOP) — — 60%
DAP (Di-Ammonium Phosphate) 18% 46% —

4.3 Fertilizer Calculation Formula

// Calculate fertilizer amount needed Fertilizer_kg = (Deficit_kg ÷ Nutrient_%) × 100 // Example: If Nitrogen deficit = 30 kg and Urea has 46% N Urea = (30 ÷ 46) × 100 = 65.22 kg Urea per acre

4.4 Scenario A: NPK Surplus

⚠ WARNING — Excess Nutrients Detected
If any nutrient in the soil exceeds the crop requirement, the system warns the farmer. Excessive nutrients can lead to environmental pollution, crop damage, and wasted resources.
Sample Response
Q: I have N=70, P=30, K=20 kg/acre in my Punjab soil. Can I grow wheat here?

A: Your soil has excess nitrogen (+10 kg/acre) compared to what wheat requires. The actual required quantity for growing wheat in your area is 60 kg/acre.
Source: ICAR - Directorate of Wheat Research (Page 8)

4.5 Scenario B: NPK Deficit — Fertilizer Combinations

When nutrients are deficient, the system calculates the required fertilizer amounts and provides multiple combination options.

Current Soil Status
  • Nitrogen (N): 35 kg/acre available
  • Phosphorus (P): 15 kg/acre available
  • Potassium (K): 8 kg/acre available
Wheat Requirement
  • Nitrogen (N): 60 kg/acre needed
  • Phosphorus (P): 24 kg/acre needed
  • Potassium (K): 16 kg/acre needed

Calculated Deficits: N: +25 kg, P: +9 kg, K: +8 kg

4.6 Combination Option A: Single Fertilizers

UREA + SSP + MOP (All Single Nutrient Fertilizers)
Urea (46% N) 55 kg/acre
Single Superphosphate (16% P₂O₅) 56 kg/acre
Muriate of Potash (60% K₂O) 13 kg/acre

4.7 Combination Option B: DAP-Based

DAP + SSP + MOP (DAP provides both N and P)
DAP (18% N, 46% P₂O₅) 20 kg/acre
Single Superphosphate (16% P₂O₅) 0 kg/acre
Muriate of Potash (60% K₂O) 13 kg/acre
Note: DAP alone covers 9 kg P₂O₅ (20 × 0.46 = 9.2 kg) ✓ and 3.6 kg N (20 × 0.18 = 3.6 kg)

4.8 Combination Option C: Urea + DAP + MOP

UREA + DAP + MOP (Alternative Mix)
Urea (46% N) 44 kg/acre
DAP (18% N, 46% P₂O₅) 20 kg/acre
Muriate of Potash (60% K₂O) 13 kg/acre
✓ Smart Exclusion
If any nutrient deficit is zero or negative (already sufficient), that fertilizer is automatically excluded from the recommendation list to avoid unnecessary costs.

5.Query Type 3: Crop Recommendations

When a farmer has soil test results but doesn't know which crops to grow. The system recommends crops suitable for their soil's nutrient profile.

5.1 Query Pattern

Searching Parameters
Location: State/region
Current Soil NPK: N, P, K values from soil test (kg/acre)

5.2 Example Query & Response

Sample Conversation
Q: My soil in Punjab has N=40, P=20, K=15 kg/acre. Which crops can I grow?

A: Based on the provided sources, you can grow these crops:

Cucumber: For cucumber, you are deficit on 0 kg/acre N, 0 kg/acre P, and 5 kg/acre K.
Source: Punjab Agricultural University Recommendations (Page 12)

Cowpea: For cowpea, you are deficit on 20 kg/acre N, 4 kg/acre P, and 5 kg/acre K.
Source: Punjab Agricultural University Recommendations (Page 15)

Onion (Kharif): For onion, you are deficit on 0 kg/acre N, 0 kg/acre P, and 5 kg/acre K.
Source: Punjab Agricultural University Recommendations (Page 18)

Note: Crops requiring higher NPK than available in soil are excluded from results.

5.3 System Flow

1
Receive Soil Data
Parse N, P, K values and location from query
2
Fetch All Crops
Get all crops for the specified state
3
Filter Suitable Crops
Keep crops where crop.N ≤ soil.N AND crop.P ≤ soil.P AND crop.K ≤ soil.K
4
Calculate Deficit
For each crop, calculate N, P, K deficit
5
Sort & Return
Sort by total deficit (lowest first) and display

6.Open Points for Discussion

⚠️ Implementation Decision Required
Question: For Query Type 3 (Crop Recommendations), should we show fertilizer requirements for each recommended crop?
Option A — Deficit Values Only:
Show only the deficit amount for each nutrient (N: +5 kg, P: +2 kg, K: +3 kg). Let the farmer decide if they want to proceed with fertilizer calculations.
Pros: Simpler response, less overwhelming
Option B — Full Fertilizer Recommendations:
Automatically calculate and show the fertilizer amounts (Urea: 11 kg, SSP: 13 kg, MOP: 5 kg) for each recommended crop.
Pros: More actionable, complete guidance
Option C — User Choice:
Show deficit values first, then ask "Would you like fertilizer recommendations for any of these crops?"
Pros: Flexible, user-controlled experience
